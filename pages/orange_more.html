Name & Image

Plant Name: Orange

Scientific Name: Citrus sinensis

Family: Rutaceae

Type: Fruit tree

2. Basic Info

Orange is a citrus fruit rich in Vitamin C, widely grown in tropical and subtropical regions. It requires well-drained soil and moderate irrigation.

3. Common Diseases

  • Citrus Canker
  • Greening Disease (HLB)
  • Root Rot
  • Sooty Mold

4. Pesticides

  • Copper fungicides
  • Imidacloprid (for HLB)
  • Neem oil sprays

5. How to Solve Issues

  • Use certified disease-free saplings.
  • Spray copper fungicide for canker control.
  • Remove infected branches promptly.
  • Control insect vectors like psyllids.
